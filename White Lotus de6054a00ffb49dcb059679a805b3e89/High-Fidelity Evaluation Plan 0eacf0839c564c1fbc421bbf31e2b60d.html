High-Fidelity Evaluation Plan

Testing Method

To test our high fidelity prototype we will perform tests on both the overall user experience and specific tasks.

For the user experience, we will have users test our high fidelity prototype and answer a System Usability Scale (SUS).

For testing our defined tasks, we will measure the efficiency and satisfaction of the users

Tasks:

  • Finding accommodation, and chat with the landlord
  • Finding insurance
  • Applying for a grant

Secondary:

  • Filtering the searches
  • Editing profile

Testing Plan

Metrics:

  1. Measure the completion rate and time each user takes for each different task, from the moment he selects the destination location until the instance he contacts the service provider (find home, insurance, grant).
  1. Measure the time it takes to register for the first time
  1. Measure the number of errors on each task to identify any unintended actions per task
  1. Measure the satisfaction of each user after performing each task individually
  1. Measure the user satisfaction with the overall app (UI, User Experience)

Goals:

  • 90% of the users could perform each task on the first try
  • Most users can register in under 2 minutes
  • Users were able to complete all tasks in less than 5 mins, starting from the moment they open the app for the first time
  • User satisfaction is at least 4/5

User Recruitment

How - Get university colleagues and friends

Who - Young adults, students

How many - 7-10 participants
