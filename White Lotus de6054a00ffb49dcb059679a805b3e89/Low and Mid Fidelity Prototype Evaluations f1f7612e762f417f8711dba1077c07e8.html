Low and Mid Fidelity Prototype Evaluations

Tasks users will perform:

  • Create a profile
  • Find a house
  • Find Health Insurance
  • Find a grant

Method:

  • Users were asked to perform a certain task (”if you wanted to [TASK], what would you do?”)
  • We asked the users to think aloud
  • The behavior of the users was observed, seeing if there were any errors or hesitation
  • At the end, the user is asked for overall feedback

Feedback for the Low-fidelity prototype:

User 1

  • “The purpose of the beginning screen seems a bit weird, it looks like a tutorial but are better ways to do one”
  • “If I already created the account and filled in my data, especially my home and host university, I should not need to fill that information again when I go search for a house/grant/insurance”
  • “However, it makes sense to search in a different city”
  • “Overall, everything is very simple and intuitive”

User 2

  • “On the main page, the squares with the tasks’ names look like buttons”
  • “There should be a way to return to the main page”
  • “It would be easier to click on the main page buttons to do the task”
  • “I Should be able to search for specific grants”

User 3

  • “Too much mandatory information when creating an account”
  • “Some profile information can’t be changed afterward”
  • “Cannot upload photo to the profile”
  • “Can’t sort or filter houses/grants/insurances ”

User 4

  • “I didn’t really understand the point of the home page”
  • “The sorting filter didn’t work”
  • “Navigated through the prototype without difficulties”

Feedback for the Mid-fidelity prototype:

User 5

  • “The “find house” button is misleading”
  • “There should be a final screen when the landlord is contacted”
  • “Have an indication that a landlord was already contacted”

User 6

  • “Should be able to see other users’ reviews of the houses and insurances”
  • “Would be useful to see the distance from the house to the host university when visiting the page of a house”
  • “The country should also be displayed when the cities are listed”

User 7:

  • “Frustrating I couldn’t log in with google or facebook like in other apps”
  • “Would be useful to have more information about the houses, like user reviews”
