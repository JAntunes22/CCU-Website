High-Fidelity Evaluation

Testing Method

To test our high fidelity prototype we will perform tests on both the overall user experience and specific tasks.

For the user experience, we will have users test our high fidelity prototype and answer a System Usability Scale (SUS).

Tasks:

  • Creating an account
  • Finding accommodation, and chat with the landlord
  • Finding insurance
  • Applying for a grant

Metrics:

  1. Measure the completion rate and time each user takes for each different task
  1. Measure the user satisfaction with the overall app (UI, User Experience) (SUS)

System Usability Scale:

1-Strongly Disagree 2-Disagree 3-Neutral 4-Agree 5-Strongly Agree

  1. I think that I would like to use this app frequently.
  1. I found the app unnecessarily complex.
  1. I thought the app was easy to use.
  1. I think that I would need the support of a technical person to be able to use this app.
  1. I found the various functions in this app were well integrated.
  1. I thought there was too much inconsistency in this app.
  1. I would imagine that most people would learn to use this app very quickly.
  1. I found the app very cumbersome to use.
  1. I felt very confident using the app.
  1. I needed to learn a lot of things before I could get going with this system.
